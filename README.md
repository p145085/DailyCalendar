# Daily Calendar

A personal daily habit tracker. No server, no install, no account.

## Usage

1. Download [`dist/index.html`](dist/index.html)
2. Open it in your browser
3. Bookmark it — that's it

All data is stored in your browser's localStorage. Nothing leaves your machine.

## Features

- Calendar view with completion indicators
- Daily task tracking (pre-loaded with Duolingo, fully customizable)
- **Today:** single-click to mark tasks done
- **Past days:** two-click confirmation to prevent accidental changes
- Progress heatmap, streaks, and completion stats
